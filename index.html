Tudo que você precisa saber sobre GIT e GITHUB

Git e GitHub são ferramentas fundamentais para o desenvolvimento de software moderno.

# Git controla versões
# GitHub hospeda repositórios online
# Juntos, tornam o trabalho em equipe mais eficiente

GIT

O Git registra cada alteração feita no projeto e permite colaboração sem sobrescrever código de outras pessoas.

Funções principais

  • Controle de versões: Rastreia cada mudança.
  • Branching: Crie linhas paralelas de desenvolvimento.
  • Commits: Registre mensagens descritivas de alterações.
  • Merge: Una branches de forma controlada.
  • Trabalho distribuído: Cada dev tem uma cópia completa.

Exemplo de uso

$ git init
$ git add .
$ git commit -m "Primeiro commit"
$ git push origin main

GITHUB

GitHub é a plataforma que centraliza os repositórios Git, permitindo colaboração global.

Funções principais

  • Pull Requests: Revisão de código antes de unir.
  • Issues: Gestão de tarefas e bugs.
  • Actions: Automação de testes e deploy.
  • Forks & Clones: Experimente sem afetar o original.

Exemplo de colaboração

# Clonar projeto
$ git clone https://github.com/usuario/repositorio.git

# Criar branch
$ git checkout -b nova-feature

# Subir alterações
$ git push origin nova-feature

Resumo

O Git é a ferramenta de versionamento, o GitHub é o espaço onde você compartilha e colabora. Juntos, são essenciais no desenvolvimento moderno.
