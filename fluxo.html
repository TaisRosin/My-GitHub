Conceitos Fluxo Git Conflitos

Fluxo do Git e GitHub

git init

Inicializa um repositório vazio, para início dos trabalhos

git clone 'url'

Criará um repositório remoto do GitHub

git remote

Mostra a URL do servidor onde o projeto está salvo

git status

Mostra status do trabalho, se possui atualização não salvas, por exemplo.

git checkout -b 'branch'

Cria uma nova branch e já vai direto pra ela

git checkout 'branch'

Vai pra essa branch já criada

git add .

Adiciona os arquivos e mudanças novas para o stage

git commit -m 'msg'

Registra uma mensagem de histórico das alterações

git push

Envia os commits ao GitHub

git pull

Sincroniza mudanças realizadas em outros dispositivos que estejam no GitHub

git merge

Mescla branches
